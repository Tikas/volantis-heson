## 下载主题

```
git clone https://github.com/heson525/volantis-heson.git themes/volantis-heson
```

## 更换主题

打开根目录`_config.yml`文件

```
theme: volantis-heson
```

## 修改主题配置

在主题目录的config.yml中找到`☆`符号

这些都是必须修改的项目，在修改的内容在里面已经说明
